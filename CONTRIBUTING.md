# Contributing

WorldWideBridges.MVP-allmight is part of the colony-brain ecosystem.

Contributions can be:

- tool additions;
- better examples;
- new templates;
- improved Apps Script automation;
- documentation improvements;
- new free-first MVP patterns;
- Skill opportunities for colony-brain.

## Contribution types

### Tool suggestion

Include:

- tool name;
- category;
- free plan status;
- connection method;
- what an LLM can operate;
- limits;
- when to use;
- when to avoid.

### Example

Add examples under `examples/`.

Each example should include:

- problem;
- free stack;
- low-cost stack;
- tool connections;
- implementation order;
- project memory update.

### New Skill opportunity

Use `references/skill-generation-protocol.md`.

Classify as:

- New Skill;
- Improvement;
- Experimental.

## Quality rules

Do not submit:

- API keys or secrets;
- unsafe automation without warnings;
- vague prompts without workflow;
- paid-only recommendations without free alternative;
- content that cannot be reused by other users;
- undocumented scripts.

## Review criteria

A contribution should be:

- useful;
- practical;
- free-first;
- clear;
- reusable;
- safe;
- easy to install or copy;
- compatible with project memory.

## Recommended process

1. Create or edit files.
2. Check `REVIEW_CHECKLIST.md`.
3. Explain what changed.
4. Submit through GitHub or through the colony-brain review flow.
