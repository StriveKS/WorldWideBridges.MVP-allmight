# Design and Video Patterns

## Design-first for non-dev validation

Use design tools when a visual prototype is faster than code.

## Free-first tools

- Canva: social posts, landing visuals, presentations, simple videos.
- Figma: UI prototypes and product flows.
- Google Slides: decks, proposals, internal diagrams.
- CapCut: short-form video editing.

## Common flows

### Landing asset flow

```text
Canva/Figma -> landing page -> form -> Sheets -> follow-up
```

### Short video validation

```text
Script -> Canva/CapCut -> short video -> social post -> lead capture
```

### Pitch deck

```text
Project history -> outline -> Google Slides/Canva -> PDF/export/share
```

## When to recommend videos/tutorials

Recommend video tutorials when the user needs to perform visual steps:

- connecting a tool;
- editing in Canva/Figma;
- publishing a page;
- configuring a domain;
- deploying visually;
- editing video.

Prefer the user's native language when possible.
