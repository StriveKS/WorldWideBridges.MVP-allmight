# Marketing and Delivery Patterns

## Minimum viable marketing

For most MVPs, start with:

```text
Landing page -> lead capture -> follow-up -> delivery tracker -> analytics
```

## Free-first stack

- Landing: Canva Sites, Google Sites, GitHub Pages or Cloudflare Pages.
- Capture: Google Forms or Tally.
- Storage: Google Sheets.
- Follow-up: Gmail + Apps Script.
- Analytics: Microsoft Clarity and/or GA4.
- Assets: Canva or Figma.

## Delivery patterns

### Manual delivery

Use when validating service/product manually.

Tools:

- Gmail;
- Google Drive;
- Docs;
- Sheets;
- Calendar.

### Semi-automated delivery

Use when repeated delivery steps appear.

Tools:

- Apps Script;
- Drive templates;
- Gmail templates;
- Sheets tracker.

### Productized delivery

Use when the MVP has repeated users or payments.

Tools:

- Supabase/Firebase;
- Stripe/Mercado Pago/Gumroad/Kiwify/Hotmart;
- email automation;
- user dashboard.

## Metrics to track early

- visits;
- form submissions;
- conversion rate;
- response time;
- delivery completion;
- user feedback;
- repeated demand.
