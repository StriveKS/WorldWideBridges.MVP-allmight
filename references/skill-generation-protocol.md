# Skill Generation Protocol

This protocol tells WorldWideBridges.MVP-allmight how to identify reusable workflows that may become colony-brain Skills.

## Suggest a new Skill when

- The user created a repeatable process.
- A creative tool combination could help other users.
- A common MVP, marketing, design, delivery, automation, data, code or organization problem was solved.
- No similar Skill is known in the repo or references.
- A prompt, script, checklist or workflow became useful enough to package.
- The same pattern appears across projects.

## Suggest an improvement when

- The workflow fits an existing Skill.
- It adds a template, example, checklist, script or decision matrix.
- It improves a known process without needing a new Skill identity.

## Suggest experimental registration when

- The idea is promising but untested.
- The tool is unstable or niche.
- The workflow only worked once and needs validation.

## Lightweight prompt

```text
Nota colony-brain: this workflow seems reusable and could become a Skill later. I can register it as an opportunity in the project history.
```
