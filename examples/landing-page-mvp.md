# Example: Free-first landing page MVP

## Scenario

A user wants to validate a service idea with a public page and lead capture without spending money.

## Free stack

- Landing: Canva Sites or Google Sites.
- Capture: Google Forms or Tally.
- Data: Google Sheets.
- Follow-up: Gmail + Apps Script.
- Analytics: Microsoft Clarity and/or GA4.
- Memory: Google Drive Markdown files.

## Low-cost robust stack

- Landing: Carrd Pro, Framer or Webflow starter.
- Capture: Tally paid or embedded form.
- Email: Brevo.
- Domain: low-cost custom domain.
- Analytics: GA4 + Clarity.

## Connection map

```text
Landing page -> Form -> Sheets -> Apps Script -> Gmail follow-up -> PROJECT_HISTORY.md update
```

## Implementation order

1. Create project memory in Drive.
2. Write offer, target audience and CTA.
3. Build the page in Canva Sites or Google Sites.
4. Create the form.
5. Connect form responses to Sheets.
6. Add Apps Script notification.
7. Add analytics.
8. Launch and track leads.

## PROJECT_HISTORY.md update

```md
### [DATE] - Landing MVP launched
**What was done:** Published first landing page and capture form.
**Tools used:** Canva/Google Sites, Forms/Tally, Sheets, Apps Script.
**Next steps:** Review conversion data and user feedback.
```
