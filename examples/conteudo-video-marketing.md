# Example: Content and short video MVP

## Scenario

A user wants to validate demand through content before building the full product.

## Free stack

- Script: LLM chat.
- Visuals: Canva.
- Editing: CapCut.
- Publishing: TikTok, Instagram, YouTube Shorts or LinkedIn.
- Capture: Tally or Google Forms.
- Storage: Google Sheets.
- Follow-up: Gmail + Apps Script.

## Low-cost robust stack

- Design/video: Canva Pro or CapCut Pro if needed.
- Email: Brevo.
- Landing: Carrd/Framer/custom domain.
- Analytics: GA4 + Clarity.

## Connection map

```text
Idea -> script -> video -> post -> form/landing -> Sheets -> follow-up -> project history
```

## Implementation order

1. Define the promise and audience.
2. Write 3 short video scripts.
3. Produce simple videos.
4. Publish with one CTA.
5. Capture interest in a form.
6. Review results.
7. Update project history.

## Metrics

- views;
- clicks;
- form submissions;
- replies/comments;
- qualified leads.
