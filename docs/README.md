# Landing page

This folder contains the static landing page for WorldWideBridges.MVP-allmight.

## Enable GitHub Pages

In the GitHub repository:

1. Open **Settings**.
2. Go to **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select branch `main`.
5. Select folder `/docs`.
6. Save.

Expected URL after activation:

```text
https://striveks.github.io/WorldWideBridges.MVP-allmight/
```
