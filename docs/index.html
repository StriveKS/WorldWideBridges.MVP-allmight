// 02 — quick start

Cinco passos.
MVP no ar.

$ open SKILL.md # 1. Copy the content into your LLM or install as a Skill   $ activate WorldWideBridges.MVP-allmight # 2. Say: "Activate WorldWideBridges.MVP-allmight for this project"   $ init project-memory --drive # 3. Create project memory in Google Drive   $ build --stack free-first # 4. Build the free-first MVP stack   $ launch ✓ MVP deployed. Memory saved. Stack documented.
