WorldWideBridges.
MVP-allmight

An open Agent Skill for creating, organizing, launching and improving MVPs with free-first tools, native connections, Google Workspace, GitHub, Apps Script, marketing flows, delivery systems and portable project memory.

Open SKILL.md GitHub repo Install guide colony-brain

What it does

Free-first MVPs

Always proposes a 100% free or near-free stack, plus a lowest-cost robust option.

Native connections

Prioritizes tools that are easy to authorize and operate through LLMs, Google Workspace, GitHub and APIs.

Project memory

Uses Markdown history files and tracking spreadsheets so projects can move across tools and LLMs without losing context.

Skill ecosystem

Detects reusable workflows that can become new Skills or improvements for colony-brain.

Quick start

1. Open SKILL.md
2. Copy the content into your LLM or install as a Skill
3. Say: "Activate WorldWideBridges.MVP-allmight for this project"
4. Create project memory in Google Drive
5. Build the free-first MVP stack

Use cases

Landing page MVP

Canva Sites or Google Sites + Forms/Tally + Sheets + Apps Script + analytics.

Internal automation

Sheets as a lightweight database, Apps Script as automation, Drive as memory.

Static app

GitHub Pages or Cloudflare Pages with feedback capture and project history.

Content validation

Short videos, landing pages, lead capture and follow-up workflows.

Part of colony-brain

colony-brain is the parent ecosystem for reusable LLM Skills, operational memory and free-first project workflows.

Projects create workflows.
Useful workflows become Skills.
Skills improve future projects.
