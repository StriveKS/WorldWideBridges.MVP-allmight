# MVP Tracking Spreadsheet Template

Recommended spreadsheet name:

```text
[PROJECT] - Acompanhamento
```

## Tabs

### 1. Visao Geral
Purpose: project snapshot.

Columns/fields:

- Project name
- Goal
- Current phase
- Current stack
- Main risk
- Next action
- Last update
